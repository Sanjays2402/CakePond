# Cake Pond 🍰

A tiny fun native macOS SwiftUI app: an ambient koi pond with glowing fish, bubbles, sparkles, and mood modes.

## Run

```bash
cd /Users/sanjay/.openclaw/workspace/CakePond
swift run -c release CakePond
```

Or open the built app bundle if present:

```bash
open /Users/sanjay/Desktop/CakePond.app
```

## Controls

- Click/tap the pond: feed sparkle koi
- `⌘F`: feed sparkles
- `⌘R`: random mood
- `⌘0`: reset
- Toggle “Orbit grid” in the top bar

## Moods

- Focus
- Chaos
- Cozy
- Party
